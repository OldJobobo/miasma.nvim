## [0.1.0] - 2026-03-26

### Added
- Initial SemVer baseline for the modular Neovim Lua colorscheme on `main`.
- Configurable theme setup with `transparent`, `term_colors`, `color_overrides`, and `highlight_overrides`.
- Highlight coverage for Treesitter, semantic tokens, diagnostics, and common Neovim plugin integrations.
- A Vim colorscheme compatibility shim via `colors/miasma.vim`.

### Changed
- Established `main` as the shipped Lua theme branch for Neovim.

### Notes
- Earlier repository history predates formal version tracking and is treated as pre-`0.1.0`.
